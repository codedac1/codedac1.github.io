부팅 자동 실행

AutoStart+

부팅하면 원하는 앱이 알아서 실행됩니다. 루트 불필요, 한 번만 설정하세요.

Google Play에서 보기 → 문의하기

개요

AutoStart+는 기기가 부팅되는 순간 선택한 앱을 자동으로 실행해주는 가장 간편한 부팅 앱 자동 실행 관리자입니다. 루트 권한이 필요 없고, 한 번만 설정하면 매번 알아서 실행됩니다. 무료 버전은 앱 1개를 자동 실행하고, PRO로 업그레이드하면 여러 앱을 원하는 순서대로 무제한 자동 실행할 수 있습니다. 자동 실행에 실패하면 알림으로 안내하며, 삼성·샤오미 등 OEM 기기의 자동 실행 설정도 앱 내에서 바로 안내합니다.

주요 기능

  • 부팅 시 선택한 앱을 즉시 자동 실행
  • 루트(Root) 권한 없이 일반 안드로이드 기기에서 작동
  • 설치된 앱을 몇 초 만에 추가하고 탭 한 번으로 활성화/비활성화
  • 자동 실행 실패 시 스마트 부팅 알림으로 안내
  • 부팅 오버레이 배너로 잠금 해제 전에도 즉시 실행 (PRO)
  • 여러 앱을 원하는 순서대로 무제한 자동 실행 (PRO)

스크린샷

스크린샷을 누르면 크게 볼 수 있습니다.

자주 묻는 질문

부팅 후 앱이 실행되지 않아요.

일부 제조사(삼성, 샤오미 등)는 자체적으로 앱 자동 실행을 차단합니다. 앱 내 안내를 따라 제조사별 '자동 실행 허용' 설정을 완료해 주세요.

몇 개의 앱을 자동 실행할 수 있나요?

무료 버전은 1개, PRO 버전은 무제한으로 자동 실행할 수 있습니다.

배터리 소모가 늘어나나요?

AutoStart+는 부팅 시 단 한 번만 동작하는 상시 실행 앱이 아니므로 배터리에 거의 영향이 없습니다.
