Clipboard+ for Windows

Pengelola clipboard tray Windows yang tersinkron dengan ponsel Android Anda melalui Google Drive Anda sendiri.

Aplikasi desktop Windows Hubungi kami

Ikhtisar

Clipboard+ for Windows adalah pengelola clipboard yang tinggal di tray dan secara otomatis merekam teks serta gambar yang Anda salin. Aplikasi ini memakai format data yang sama dengan aplikasi Android Clipboard+, sehingga ponsel dan PC Anda tersinkron dua arah melalui Google Drive Anda sendiri (folder tersembunyi khusus aplikasi) tanpa server khusus. Aplikasi menangkap klip lewat event clipboard Windows native tanpa polling, dan memungkinkan Anda merapikan teks serta gambar ke dalam grup dan snippet lalu mencarinya.

Fitur Utama

  • Tinggal di tray dengan pemantauan clipboard real-time tanpa polling (event Win32), menangkap otomatis teks dan gambar
  • Sinkronisasi dua arah ponsel-ke-PC melalui Google Drive Anda sendiri — tanpa server khusus, hanya memakai scope drive.appdata
  • UI bergaya Android dengan tema terang/gelap, sidebar grup, snippet, dan pin/salin/edit/cari
  • Pindah, hapus, dan gabung massal multi-pilih, urutan terbaru/terlama, dan penampil gambar ukuran penuh
  • Hotkey global untuk membuka jendela dan langsung menangkap teks yang dipilih di aplikasi lain, plus jalan otomatis saat startup
  • Ekspor/impor kompatibel dengan format cadangan Android Clipboard+ (teks JSON, gambar ZIP)

FAQ

Apakah tersinkron dengan aplikasi Android Clipboard+?

Ya. Kedua aplikasi memakai format sinkronisasi yang sama dan menjaga clipboard ponsel serta PC Anda tersinkron dua arah melalui folder khusus aplikasi di Google Drive Anda sendiri, tanpa server khusus yang terlibat.

Di mana data clipboard saya disimpan, dan apakah aman?

Data lokal disimpan di %APPDATA%\ClipboardPlus pada PC Anda, dan data yang tersinkron hanya berada di area tersembunyi khusus aplikasi di Google Drive Anda sendiri. Aplikasi hanya memakai scope drive.appdata, sehingga tidak pernah menyentuh file Drive biasa Anda.

Apa yang saya butuhkan untuk mengaktifkan sinkronisasi?

Untuk mengaktifkan sinkronisasi Google Drive, buat satu kali klien OAuth desktop di Google Cloud Console, letakkan file google_client.json di folder aplikasi, lalu klik sinkronisasi di aplikasi untuk melakukan login Google satu kali dan memberikan izin.
