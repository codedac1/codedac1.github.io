SecretAlbum+

Sembunyikan dan lindungi foto serta video Anda dengan enkripsi AES-256 dan kunci PIN.

Lihat di Google Play → Hubungi kami

Ikhtisar

SecretAlbum+ adalah brankas pribadi yang melindungi foto dan video berharga Anda dengan enkripsi AES-256 tingkat militer. Aplikasi bekerja sepenuhnya offline di perangkat Anda dan tidak pernah mengunggah file ke server eksternal mana pun. Setiap file yang Anda impor langsung dienkripsi dan tidak bisa diakses tanpa kunci pribadi Anda, hilang sepenuhnya dari galeri tanpa meninggalkan jejak. Anda bahkan bisa menyamarkan ikon aplikasi menjadi aplikasi Kalkulator, Catatan, Cuaca, atau Jam sehingga tak ada yang tahu aplikasi ini terpasang.

Fitur Utama

  • Enkripsi AES-256 tingkat militer langsung melindungi setiap file
  • Kunci PIN 6 digit dengan pemulihan PIN sementara via email setelah 5 kali gagal
  • Mode penyamaran mengubah ikon menjadi aplikasi Kalkulator, Catatan, Cuaca, atau Jam
  • Impor banyak foto dan video dari galeri, dengan aslinya otomatis dihapus
  • Penampil layar penuh dengan cubit-untuk-zoom dan tayangan slide otomatis
  • Berfungsi 100% offline dengan pemulihan tanpa kehilangan kualitas kembali ke galeri kapan saja

Tangkapan Layar

Ketuk tangkapan layar untuk melihatnya lebih besar.

FAQ

Apakah file saya dikirim ke server atau cloud eksternal?

Tidak. SecretAlbum+ bekerja sepenuhnya offline dan tidak pernah terhubung ke server eksternal mana pun. Semua file hanya disimpan di ruang terenkripsi pada perangkat Anda.

Apa yang terjadi jika saya lupa PIN?

Setelah 5 kali salah memasukkan PIN 6 digit berturut-turut, PIN sementara sekali pakai otomatis dikirim ke alamat pemulihan terdaftar Anda sehingga Anda tidak pernah terkunci permanen.

Bisakah saya memindahkan file tersembunyi kembali ke galeri?

Bisa. Anda bisa memulihkan file apa pun ke galeri kapan saja, dan proses dekripsinya tanpa kehilangan kualitas, sehingga setiap foto dan video kembali persis seperti aslinya tanpa penurunan kualitas.
