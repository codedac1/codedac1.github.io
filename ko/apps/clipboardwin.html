Clipboard+ for Windows

안드로이드와 클립보드를 내 구글 드라이브로 잇는 Windows 트레이 클립보드 매니저

Microsoft Store에서 보기 → 문의하기

개요

Clipboard+ for Windows는 시스템 트레이에 상주하며 복사한 텍스트와 이미지를 자동으로 기록하는 클립보드 매니저입니다. 안드로이드 Clipboard+ 앱과 동일한 데이터 구조를 사용해, 별도 서버 없이 사용자 본인의 구글 드라이브(앱 전용 숨김 영역)를 통해 폰과 PC의 클립보드를 양방향으로 동기화합니다. 폴링 없이 Windows 클립보드 이벤트를 감지해 캡처하며, 텍스트와 이미지를 그룹·스니펫으로 정리하고 검색할 수 있습니다.

주요 기능

  • 트레이 상주 + 폴링 없는 실시간 클립보드 감시(Win32 이벤트), 텍스트·이미지 자동 캡처
  • 내 구글 드라이브를 통한 폰↔PC 양방향 동기화(전용 서버 없음, drive.appdata 스코프만 사용)
  • 안드로이드 스타일 UI — 라이트/다크 테마, 그룹 사이드바, 스니펫, 핀·복사·편집·검색
  • 다중 선택 일괄 이동/삭제/병합, 최신·오래된순 정렬, 이미지 원본 뷰어
  • 전역 단축키로 창 열기 및 타 앱에서 선택한 텍스트 바로 캡처, Windows 자동 실행
  • 안드로이드 Clipboard+ 백업 포맷 호환 내보내기/가져오기(텍스트 JSON, 이미지 ZIP)

자주 묻는 질문

안드로이드 Clipboard+ 앱과 동기화되나요?

네. 두 앱이 동일한 동기화 포맷을 공유하며, 사용자 본인의 구글 드라이브 앱 전용 폴더를 통해 폰과 PC의 클립보드가 양방향으로 동기화됩니다. 별도의 서버는 사용하지 않습니다.

제 클립보드 데이터는 어디에 저장되나요? 안전한가요?

로컬 데이터는 PC의 %APPDATA%\ClipboardPlus 폴더에 저장되고, 동기화 데이터는 오직 사용자 본인 구글 드라이브의 앱 전용 숨김 영역에만 저장됩니다. 앱은 drive.appdata 스코프만 사용하므로 사용자의 일반 드라이브 파일에는 접근하지 않습니다.

동기화를 사용하려면 무엇이 필요한가요?

구글 드라이브 동기화를 켜려면 구글 클라우드 콘솔에서 데스크톱용 OAuth 클라이언트를 한 번 발급받아 google_client.json 파일을 앱 폴더에 넣고, 앱에서 동기화 버튼을 눌러 최초 1회 구글 로그인/권한 동의를 하면 됩니다.
